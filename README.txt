A Retro a PC/XT emulator (http://www.xs4all.nl/~rjoris/retro/) fork for testing new ideas and algorithms in VMs and emulators.

Features
--------

Implemented:
 * 8086/8088 CPU
 * Color Graphics Adapter (CGA)
 * I8254 timer
 * I8259 interrupt controller
 * I8237 DMA controller
 * keyboard controller
 * floppy drive(s)
 * custom BIOS code (minimal, but sufficient for booting DOS etc)
